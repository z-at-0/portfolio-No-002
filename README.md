# portfolio-No-002: Gallery Site

## 概要
このサイトは画像を閲覧できるギャラリーサイトとして作成しました。  
写真やイラストなどの作品を直感的に楽しめることを意識して設計しています。

## 主な機能
トップページでのシームレスなナビゲーション
カテゴリ別に作品を絞り込める動的なギャラリー設計
モーダルウィンドウによる画像の拡大・詳細表示

## 使用技術

使用言語: HTML5, CSS3, JavaScript
技術要素: JavaScriptによるDOM操作とイベント制御

## 設計方針

ユーザーの没入感を損なわずに画像の世界観に浸れるよう、装飾を抑えたシンプルなデザインを採用しました。
  
## 学習・AI活用について

本プロジェクトは2021年に制作されたものであり、AIツール等は一切使用していません。

■ 補足

004〜006に至る学習の初期段階として、「JavaScriptでいかにブラウザ上の要素を自由に操るか」という基礎を固めるために制作したプロジェクトです。この時期にスクリプトを自力で書き抜いた経験が、現在の高度な非同期処理やAPI連携の実装を支える土台となっています。
